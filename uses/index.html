What I'm using right now

4 minutes read

This page lists the tools, apps, and services that I use. I got inspired to write this after seeing Kent C. Dodds’ list of tools he uses.

Here’s what each of the labels mean:

  • ❤️ Can’t imagine using anything else.
  • 👍 Does the job well. No plans on switching.
  • 🤷‍♂️ Would prefer to find a better alternative.

Desktop apps

  • ❤️ VS Code with the Enough editor theme.
  • ❤️ Raycast has replaced Alfred for me.
  • ❤️ Obsidian
  • ❤️ ️️1Password
  • ❤️ Mullvad VPN
  • ❤️ Signal
  • ❤️ Affinity Designer — An excellent (and cheap!) alternative to Adobe Illustrator.
  • ❤️ CleanShot X
  • 👍 Firefox, but I use Brave for anything that needs Blink.
  • 👍 Camtasia for screen recording and video productions.
  • 👍 Pixelmator Pro — For everyday image editing. I mostly use Affinity Designer most of the time though.
  • 👍 Little Snitch
  • 👍 Spotify
  • 🤷‍♂️ Audacity for when I need to record my voice.
  • 🤷‍♂️ Fantastical — Trying it out for a year. So far I’m not sure how it’s any better than the macOS Calendar app.
  • 🤷‍♂️ Slack
  • 🤷‍♂️ Discord
  • 🤷‍♂️ Zoom

Services

  • ❤️ You Need A Budget — Wish I found this when I got my first job. Now I know where my money goes.
  • ❤️ Headspace — Probably the biggest contributor to my mental health.
  • 👍 Tresorit — End-to-end encrypted cloud storage with a decent mobile app. Only downside is the price.
  • 👍 Buy Me A Coffee — For projects I’m working on outside my day-job. Honestly, it’s more for managing my imposter syndrome than to generate income.
  • 👍 LanguageTool — Not quite as good as Grammarly, but I prefer their privacy policy.
  • 🤷‍♂️ Fastmail, HEY, ProtonMail, and GMail — My email is a mess. I love the HEY model, but it’s a bit pricey and has no calendar support. Right now I’m evaluating Fastmail.
  • 🤷‍♂️ Pocket — Not a fan, but better than any of the alternatives.

Website

  • ❤️ GitHub Pages using Hugo
  • ❤️ Plausible — Check out the public dashboard.
  • ❤️ updown.io — Check out the public dashboard.
  • 👍 DNSimple

Tools and tech

  • ❤️ Go is my primary programming language.
  • ❤️ Docusaurus for anything that needs proper documentation.
  • 👍 TypeScript for anything that needs to run in the browser.
  • 👍 React for more complex web applications, though I’m excited about Svelte.
  • 👍 rescript for late night functional programming.
  • 👍 brew
  • 👍 ray.so has replaced carbon for code screenshots.

Desk setup

  • 🤷‍♂️ MacBook Pro (13-inch, 2019) — The model from before Apple realized they messed up.
  • ❤️ ZSA Moonlander with Kailh Copper switches — Best keyboard I’ve experienced so far.
  • ❤️ Logitech MX Vertical — I keep this between the Moonlander splits.
  • ❤️ HÅG Capisco — I shift around a lot and this desk chair is still comfortable.
  • ❤️ YETI Rambler 14 oz Mug — Holds my liquid black gold during the day.
  • 👍 Grovemade Wool Felt Desk Pad — Looks good, but also keeps the Moonlander in place as it does tend to slide away from me otherwise.
  • 👍 twelve south Curve — Elevates my MacBook from my desk.
  • 👍 Dell UltraSharp U2718Q 27" — I use this as my primary display with the MacBook on the side.
  • 👍 Multibrackets M Deskmount Basic Single — Holds my primary monitor.
  • 👍 Jarvis Standing Desk Frame with a BEKANT table top — Reused a table top I already had, but I dream about replacing it with a slab of walnut wood one day.
  • 👍 JobOut Standing Mat Classic — I swap the chair out for this when I’m standing.

Audio and video

  • ❤️ Sony Alpha a6400 Mirrorless Digital Camera — I use this as my webcam for webinars, daily check-ins, and other calls.
  • 🤷‍♂️ Sigma 30mm f/1,4 DC DN — Bought this lens with the camera, but I’m not quite happy with the field of view. Looking at the 16mm version.
  • ❤️ Elgato Cam Link 4K — Lets me use my camera as a USB webcam.
  • 👍 Elgato Master Mount — For mounting the camera behind the monitor.
  • 👍 Sony ACPW20 AC Adaptor — powers the camera so that I don’t need to charge any batteries.
  • ❤️ Elgato Key Light — I have two of these, but I tend to only use one during daily calls. Basically only turn on both when I’m doing webinars or other recordings.
  • 👍 RODE NT-USB with the RODE PSA1 Arm, and RODE PSM1 Shock Mount.
  • 👍 Sennheiser HD XXX — Headphones for video calls. I bought these cheap and have no idea what model they are.
  • 👍 6amLifestyle Headset Holder — Holds my Sennheisers when I’m not in a call.
  • 👍 Sony WH-1000XM4 — Headphones for when I travel.

Travel gear

  • ❤️ Pakt Travel Backpack — For longer travels.
  • ❤️ Patagonia Atom Sling 8L — For when I don’t need my laptop. Pairs well with the Pakt Travel Backpack when I’m flying.
  • 👍 Aer Cable Kit — This keeps all my cables and adapters in one place.
  • 👍 Aer Day Pack — For commuting.
  • 👍 Kindle Paperwhite (7th Generation) — Still holds up!
  • 👍 Logitech R500s Laser Presentation Remote — Mostly using this for in-person conference talks.
